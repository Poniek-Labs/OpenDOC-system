## What OpenDOC Is

OpenDOC is an open protocol for rich documents with:

- Human-inspectable metadata
- Portable ZIP-based packaging
- HTML content for rich text compatibility
- Explicit manifest entries for reproducible assets

## Core Terms

- `OpenDOC`: Protocol and ecosystem name
- `.sodf`: Source OpenDOC Format file extension
- `Interpreter`: High-level runtime API for apps and tools

## Design Principles

- Open first: no closed binary internals
- Predictable: strict required paths and schema validation
- Extensible: versioned metadata and styles
- Language-neutral format with a Python reference implementation
